# Tables

1|                  | ASCII                           | HTML                          |
2| ---------------- | ------------------------------- | ----------------------------- |
3| Single backticks | `'Isn't this fun?'`             | 'Isn't this fun?'             |
4| Quotes           | `"Isn't this fun?"`             | "Isn't this fun?"             |
5| Dashes           | `-- is en-dash, --- is em-dash` | -- is en-dash, --- is em-dash |

Colons can be used to align columns.

1| Tables        |      Are      | Cool |
2| ------------- | :-----------: | ---: |
3| col 3 is      | right-aligned |      |
4| col 2 is      |   centered    |      |
5| zebra stripes |   are neat    |

col 3 isright-aligned
col 2 iscentered
zebra stripesare neat

The outer pipes (|) are optional, and you don’t need to make the raw Markdown line up prettily. You can also use inline Markdown.
